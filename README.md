### top
    - Linux에서 실행 중인 프로세스의 실시간을 표시하고 커널 관리 작업을 표시한다.
이 명령어는 CPU 및 메모리 사용량을 포함한 리소스 사용량을 보여주는 시스템 정보 요약을 제공한다.
윈도우의 작업관리자랑 비슷하다.
